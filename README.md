# 🎮 Loja de Games – Java

Projeto final do Bloco 01 do **Bootcamp Fullstack Java – Generation Brasil**.

Sistema de gerenciamento de uma loja de games desenvolvido em Java, aplicando os fundamentos de Programação Orientada a Objetos e CRUD completo via terminal.

---

## 🚀 Funcionalidades

- ✅ Cadastrar produto (game)
- ✅ Listar todos os produtos
- ✅ Buscar produto por ID
- ✅ Atualizar produto
- ✅ Deletar produto
- ✅ Menu interativo com cores ANSI no terminal

## 📁 Estrutura do projeto

```

└── src/
    ├── model/        # Classes de modelo (Produto/Game)
    ├── repository/   # Lógica de armazenamento e busca
    └── Menu.java     # Menu principal e interação com usuário
```

## ▶️ Como executar

1. Clone o repositório:
```bash
git clone https://github.com/kauedota/projeto_final_bloco_01.git
```

2. Abra o projeto na sua IDE (IntelliJ, Eclipse ou VS Code)

3. Execute o arquivo `Menu.java`
